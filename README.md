# Key in the Door Web

Immersive dark web experience for the concept album project "Za Drzwiami / Pętla Cienia". This project is a narrative-driven, interactive experience that explores themes of trauma, recovery, and self-discovery. The website is designed to resemble a dark, atmospheric, and immersive environment, with a focus on storytelling and user interaction.

## Features
- Interactive narrative with 6 threshold sections and an ending
- Immersive dark web design with a focus on typography and color scheme
- Dynamic i18n implementation with support for Polish and English languages
- IntersectionObserver for scroll-triggered animations
- ARIA labels and accessibility features for improved usability

## Technologies Used
- HTML5
- CSS3
- JavaScript (for dynamic i18n and IntersectionObserver)
